2. 작업 환경 만들기
(1)Azure portal 좌측상단의 '리소스 만들기(+)'를 클릭합니다.
(2) 좌측의 범주 배너에서 'AI+기계 학습'을 클릭합니다.
(3) 'Azure AI services'를 클릭합니다.
(4) 리소스 이름 등등 짓고 가격 책정하기
(5) '검토 + 만들기' 클릭 후 대기합니다.
(6) '만들기' 버튼을 클릭합니다.
(7) '리소스로 이동'을 클릭합니다.
(8) 'Document Intelligent'를 선택합니다.
(9) 'Document Intelligent Studio'의 '스튜디오로 이동'을 클릭합니다.
